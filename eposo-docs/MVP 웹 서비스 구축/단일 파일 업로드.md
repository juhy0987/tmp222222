# 단일 파일 업로드

<h3>Objective</h3><p>사용자가 로컬 PC의 파일을 안정적이고 빠르게 클라우드 스토리지에 저장할 수 있도록 한다.</p><h3>Key Results</h3><ul><li>100MB 파일 업로드 성공률 99.9% 달성</li><li>업로드 속도 사용자 네트워크 대역폭의 80% 이상 활용</li><li>업로드 중 브라우저를 닫았다가 다시 열었을 때, 업로드를 이어 할 수 있는 기능(Resumable Upload) 지원</li></ul><h3>기능 명세</h3><ul><li><b>업로드 인터페이스:</b> 사용자는 '업로드' 버튼을 클릭하여 파일 탐색기를 열거나, 파일을 브라우저 창으로 직접 드래그 앤 드롭하여 업로드를 시작할 수 있다. 드래그 앤 드롭 시, 드롭 가능한 영역이 점선 테두리와 함께 시각적으로 강조 표시된다.</li><li><b>업로드 프로세스:</b> 파일이 선택되면, 화면 우측 하단에 업로드 진행 상태를 보여주는 패널이 나타난다. 이 패널에는 파일명, 전체 용량 대비 업로드된 용량(MB), 진행률(%), 남은 시간이 표시된다. 20MB 이상의 파일은 자동으로 5MB 단위의 청크(Chunk)로 분할되어 병렬로 전송되어 안정성과 속도를 높인다. 업로드 중 네트워크 연결이 끊기거나 브라우저가 종료될 경우, 재연결 시 중단된 청크부터 업로드를 재개한다.</li><li><b>제한 및 오류 처리:</b> MVP 단계에서는 개별 파일의 최대 업로드 용량을 1GB로 제한한다. 용량 초과 시 "업로드 가능한 최대 파일 크기는 1GB입니다."라는 오류 메시지를 즉시 표시한다. 업로드 실패 시, 해당 파일 옆에 '재시도' 버튼을 제공한다.</li></ul>
