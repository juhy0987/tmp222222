# 파일 목록 보기 및 다운로드

<h3>Objective</h3><p>사용자가 자신이 업로드한 파일들을 명확하게 인지하고, 필요할 때 즉시 다운로드할 수 있는 기본적인 파일 관리 환경을 제공한다.</p><h3>Key Results</h3><ul><li>1000개 파일이 있는 폴더의 목록 로딩 시간 2초 미만</li><li>파일 다운로드 시작까지의 평균 대기 시간 1초 미만</li><li>표시되는 파일 메타데이터(용량, 날짜)의 정확도 100%</li></ul><h3>기능 명세</h3><ul><li><b>파일 목록 UI:</b> 파일과 폴더는 테이블 형태의 리스트로 표시된다. 각 행은 파일 유형 아이콘, 파일명, 마지막 수정일, 파일 크기 정보를 포함하는 컬럼으로 구성된다. 사용자는 각 컬럼의 헤더를 클릭하여 해당 기준(이름순, 날짜순, 크기순)으로 목록을 오름차순/내림차순 정렬할 수 있다. 한 페이지에는 기본적으로 50개의 항목을 표시하며, 목록 하단에는 페이지네이션 컨트롤을 제공하여 다른 페이지로 이동할 수 있도록 한다.</li><li><b>다운로드 기능:</b> 사용자가 파일명을 클릭하면 해당 파일의 다운로드가 즉시 시작된다. 브라우저는 서버로부터 `Content-Disposition: attachment` 헤더를 받아 파일을 실행하는 대신 저장하도록 유도한다. 여러 파일을 다운로드할 수는 없으며, 한 번에 하나의 파일만 다운로드 가능하다. 폴더 자체는 다운로드할 수 없으며, 클릭 시 해당 폴더 내부로 진입한다.</li></ul>
