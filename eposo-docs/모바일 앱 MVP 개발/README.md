# 모바일 앱 MVP 개발

<h3>산출물</h3><ul><li>iOS 및 Android 플랫폼에서 동작하는 네이티브 모바일 애플리케이션</li><li>앱 스토어(App Store, Play Store) 심사 제출이 가능한 수준의 빌드 파일(.ipa, .apk)</li><li>모바일 환경에서의 파일 조회, 업로드, 다운로드, 미리보기 기능 구현</li></ul><h3>완료 체크리스트</h3><ul><li>[ ] 기존 웹 계정으로 모바일 앱 로그인이 가능한가?</li><li>[ ] 모바일 기기의 갤러리에서 사진이나 동영상을 선택하여 업로드할 수 있는가?</li><li>[ ] 모바일 화면에 최적화된 UI로 파일 및 폴더 목록을 스크롤하며 탐색할 수 있는가?</li><li>[ ] 앱 내에서 이미지를 전체 화면으로 보고, 동영상을 스트리밍으로 재생할 수 있는가?</li><li>[ ] 파일 공유 링크를 생성하여 카카오톡, 문자 메시지 등 다른 앱으로 전달할 수 있는가?</li><li>[ ] 앱이 백그라운드로 전환되었다가 다시 포그라운드로 돌아와도 상태가 유지되는가?</li></ul>
