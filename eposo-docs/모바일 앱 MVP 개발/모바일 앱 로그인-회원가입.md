# 모바일 앱 로그인/회원가입

<h3>Objective</h3><p>모바일 사용자가 웹과 동일한 계정으로 서비스를 이용하고, 신규 가입도 앱 내에서 끊김 없이 진행할 수 있도록 한다.</p><h3>Key Results</h3><ul><li>앱 실행 후 로그인 화면 표시까지 2초 미만</li><li>로그인 API 요청 후 메인 화면 진입까지 평균 1.5초 소요</li><li>생체 인증(Face ID, 지문)을 통한 로그인 성공률 99.5% 이상</li></ul><h3>기능 명세</h3><ul><li><b>UI/UX:</b> 앱 실행 시 로고 스플래시 화면을 1.5초간 표시한 후, 로그인 화면으로 전환한다. 이메일과 비밀번호 입력 필드는 터치 키보드에 가려지지 않도록 화면 레이아웃을 동적으로 조정한다. 입력 필드 하단에는 '로그인', '회원가입', '비밀번호 찾기' 링크와 함께 Google, Apple 소셜 로그인 버튼을 배치하여 접근성을 높인다.</li><li><b>로그인 기능:</b> 웹에서 생성한 계정으로 로그인이 가능해야 한다. 로그인 성공 시, 서버로부터 발급받은 Access Token과 Refresh Token을 기기 내 안전한 저장소(iOS: Keychain, Android: Keystore)에 암호화하여 저장한다. '자동 로그인' 옵션을 기본으로 활성화하여 다음 앱 실행 시 별도 과정 없이 메인 화면으로 진입시킨다.</li><li><b>회원가입 기능:</b> '회원가입' 버튼 터치 시, 웹뷰를 띄우는 대신 모바일 환경에 최적화된 네이티브 UI로 회원가입 절차를 진행한다. 각 입력 필드의 유효성 검사는 실시간으로 이루어진다.</li><li><b>생체 인증:</b> 최초 로그인 성공 후, "생체 인증을 사용하여 더 빠르고 안전하게 로그인하시겠습니까?"라는 팝업을 통해 사용자 동의를 얻어 기능을 활성화한다. 이후 앱 실행 시 생체 인증을 우선적으로 시도한다.</li></ul>
