# 모바일 사진/동영상 업로드

<h3>Objective</h3><p>모바일 사용자가 스마트폰으로 촬영한 사진이나 동영상을 앱을 통해 쉽고 빠르게 클라우드에 업로드할 수 있도록 한다.</p><h3>Key Results</h3><ul><li>모바일 갤러리 접근 및 사진 선택 후 업로드 시작까지 3초 이내</li><li>백그라운드 업로드 지원 및 앱 종료 후에도 업로드 큐 유지</li><li>Wi-Fi 환경에서만 업로드하는 옵션 제공으로 사용자 데이터 90% 이상 절약</li></ul><h3>기능 명세</h3><ul><li><b>업로드 플로우:</b> 메인 화면의 플로팅 액션 버튼(FAB)을 터치하면 '사진/동영상 업로드' 옵션이 나타난다. 이를 선택하면 OS의 기본 이미지 피커(Image Picker)가 실행되어 사용자가 갤러리에서 하나 또는 여러 개의 항목을 선택할 수 있다. 선택 완료 후, 업로드할 파일 목록 확인 화면을 거쳐 업로드를 시작한다.</li><li><b>백그라운드 업로드:</b> 업로드는 백그라운드 서비스로 동작하여, 사용자가 다른 앱을 사용하거나 화면을 꺼도 중단되지 않는다. 업로드 진행 상태는 OS의 알림(Notification) 영역에 표시되며, 진행률과 남은 파일 개수를 확인할 수 있다.</li><li><b>네트워크 관리:</b> 앱 설정 메뉴에 'Wi-Fi에서만 업로드' 옵션을 제공하고 기본값으로 활성화한다. 이 옵션이 켜져 있을 때 셀룰러 데이터 환경에서는 업로드가 자동으로 일시 중지되고, Wi-Fi에 연결되면 자동으로 재개된다. 사용자에게 데이터 사용에 대한 선택권을 부여하여 만족도를 높인다.</li><li><b>자동 올리기 (Camera Uploads):</b> 사용자가 동의할 경우, 스마트폰으로 새로 촬영하는 모든 사진과 동영상을 지정된 폴더에 자동으로 업로드하는 기능을 제공한다.</li></ul>
