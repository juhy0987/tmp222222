# 모바일 파일 탐색기 UI

<h3>Objective</h3><p>모바일 기기의 작은 화면에서도 사용자가 파일과 폴더를 쉽고 편리하게 탐색하고 관리할 수 있는 최적화된 UI/UX를 제공한다.</p><h3>Key Results</h3><ul><li>폴더 진입 및 파일 목록 스크롤 시 초당 60프레임(fps) 유지</li><li>파일/폴더에 대한 주요 액션(이름 변경, 삭제 등)을 3번 이하의 터치로 수행 가능</li><li>사용자 만족도 조사에서 탐색 편의성 항목 5점 만점에 4.2점 이상 획득</li></ul><h3>기능 명세</h3><ul><li><b>레이아웃:</b> 파일과 폴더는 리스트 뷰(List View)와 그리드 뷰(Grid View) 두 가지 형태로 볼 수 있는 전환 버튼을 제공한다. 리스트 뷰는 파일명, 수정일, 용량 정보를, 그리드 뷰는 이미지와 동영상의 썸네일을 중심으로 표시한다. 화면 상단에는 현재 폴더 경로를 나타내는 브레드크럼(Breadcrumb)을 표시하여 쉽게 상위 폴더로 이동할 수 있도록 한다.</li><li><b>인터랙션:</b> 폴더를 터치하면 해당 폴더로 진입하며, 화면 전환 애니메이션은 0.3초 이내에 완료된다. 파일/폴더를 길게 누르면(Long Press) 다중 선택 모드로 전환되며, 사용자는 여러 항목을 선택하여 일괄 작업을 수행할 수 있다. 각 항목 옆에는 '더보기(...)' 아이콘을 두어 이름 변경, 이동, 공유, 삭제 등의 추가 옵션을 제공하는 Bottom Sheet UI를 표시한다. 화면을 아래로 당기면(Pull-to-Refresh) 현재 목록을 새로고침하는 기능을 제공한다.</li><li><b>오프라인 지원:</b> 사용자가 '오프라인 사용'으로 표시한 파일은 로컬에 캐싱하여, 네트워크 연결이 없을 때도 내용을 확인할 수 있도록 한다.</li></ul>
