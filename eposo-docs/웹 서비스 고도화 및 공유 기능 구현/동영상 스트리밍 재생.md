# 동영상 스트리밍 재생

<h3>Objective</h3><p>사용자가 대용량 동영상 파일을 다운로드하지 않고도 웹 브라우저에서 즉시 재생하여 콘텐츠를 빠르게 소비할 수 있도록 한다.</p><h3>Key Results</h3><ul><li>1080p 해상도 동영상 재생 시작(첫 프레임 로딩)까지 평균 3초 이내</li><li>재생 중 버퍼링으로 인한 멈춤 시간, 전체 재생 시간의 1% 미만</li><li>주요 브라우저(Chrome, Safari, Firefox, Edge) 최신 버전과의 호환성 100%</li></ul><h3>기능 명세</h3><ul><li><b>비디오 플레이어 UI:</b> 파일 목록에서 MP4, MOV 등 지원되는 동영상 파일을 클릭하면, 라이트박스(Lightbox) 형태의 비디오 플레이어 모달이 화면 중앙에 나타난다. 플레이어는 재생/일시정지 버튼, 진행 상태를 보여주는 타임라인 바, 현재 시간/전체 시간 표시, 음량 조절 슬라이더, 재생 속도 조절(0.5x, 1x, 1.5x, 2x), 전체화면 전환 버튼을 포함한다.</li><li><b>스트리밍 기술:</b> 서버는 HTTP Live Streaming (HLS) 또는 MPEG-DASH 프로토콜을 사용하여 동영상을 서비스한다. 원본 동영상은 업로드 시 다양한 해상도(360p, 720p, 1080p)의 세그먼트로 미리 인코딩된다. 클라이언트 플레이어는 사용자의 네트워크 대역폭을 실시간으로 감지하여 최적의 해상도를 자동으로 선택하는 적응형 비트레이트 스트리밍(Adaptive Bitrate Streaming)을 구현하여 버퍼링을 최소화한다. 사용자가 타임라인의 특정 지점을 클릭하면, 해당 시간대의 세그먼트부터 즉시 다운로드하여 재생을 시작한다.</li></ul>
