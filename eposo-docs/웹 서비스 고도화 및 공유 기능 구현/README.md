# 웹 서비스 고도화 및 공유 기능 구현

<h3>산출물</h3><ul><li>폴더 생성/삭제/이동, 다중 파일 다운로드, 동영상 스트리밍, 파일 공유 링크 생성이 추가된 웹 애플리케이션</li><li>클로즈 베타 테스트(CBT)를 진행할 수 있는 수준의 안정화된 웹 서비스</li><li>주요 기능에 대한 사용자 시나리오 기반의 QA 테스트 결과서</li></ul><h3>완료 체크리스트</h3><ul><li>[ ] 사용자가 새로운 폴더를 생성하고 이름을 변경하며 삭제할 수 있는가?</li><li>[ ] 파일을 드래그 앤 드롭하여 다른 폴더로 이동시킬 수 있는가?</li><li>[ ] 2개 이상의 파일을 선택하여 하나의 ZIP 파일로 압축 다운로드할 수 있는가?</li><li>[ ] MP4 형식의 동영상 파일을 클릭 시, 별도 다운로드 없이 즉시 스트리밍 재생이 시작되는가?</li><li>[ ] 특정 파일에 대해 '보기 전용' 공유 링크를 생성하고, 해당 링크로 접속 시 파일 내용 확인이 가능한가?</li><li>[ ] 모든 기능이 주요 브라우저(Chrome, Safari, Edge) 최신 버전에서 동일하게 동작하는가?</li></ul>
