# 파일 공유 링크 생성

<h3>Objective</h3><p>사용자가 자신의 파일을 특정인 또는 불특정 다수에게 간편하게 전달하고 공유할 수 있는 핵심 기능을 제공한다.</p><h3>Key Results</h3><ul><li>공유 링크 생성 완료까지 1초 이내</li><li>생성된 링크의 유효성 및 접근성 100% 보장</li><li>공유 설정 UI의 사용자 이해도 90% 이상 (별도 가이드 없이 사용 가능)</li></ul><h3>기능 명세</h3><ul><li><b>링크 생성:</b> 사용자는 파일의 컨텍스트 메뉴 또는 파일 선택 시 나타나는 상단 액션 바에서 '공유' 버튼을 클릭한다. 클릭 시 공유 설정 모달이 나타나며, "링크가 있는 모든 사용자가 볼 수 있음" 권한으로 설정된 고유 URL이 자동으로 생성된다. 사용자는 '링크 복사' 버튼을 클릭하여 URL을 클립보드에 쉽게 복사할 수 있다.</li><li><b>권한 관리:</b> MVP 단계에서는 '보기 전용(Viewer)' 권한만 제공한다. 링크를 통해 접근한 사용자는 파일을 보거나 다운로드할 수는 있지만, 수정, 삭제, 이름 변경 등은 할 수 없다.</li><li><b>공유 상태 표시:</b> 파일 목록에서 공유 링크가 생성된 파일은 파일명 옆에 링크 아이콘을 표시하여 다른 파일과 시각적으로 구분할 수 있도록 한다. 이 아이콘에 마우스를 올리면 "이 파일은 링크로 공유 중입니다"라는 툴팁이 나타난다. 사용자는 언제든지 공유 모달을 다시 열어 '공유 중지' 버튼을 통해 링크를 비활성화할 수 있으며, 비활성화된 링크는 즉시 접근이 차단된다.</li></ul>
