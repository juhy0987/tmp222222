# 파일 및 폴더 이동

<h3>Objective</h3><p>사용자가 파일과 폴더의 위치를 드래그 앤 드롭(Drag and Drop)과 같은 직관적인 방식으로 쉽게 변경하여 저장 공간을 효율적으로 구성할 수 있도록 한다.</p><h3>Key Results</h3><ul><li>단일 파일/폴더 이동 작업의 UI 반응 시간 300ms 미만</li><li>10개 이상의 다중 항목 이동 시 작업 완료까지 2초 이내</li><li>이동 작업 중 데이터 유실 발생률 0%</li></ul><h3>기능 명세</h3><ul><li><b>드래그 앤 드롭 인터페이스:</b> 사용자는 파일 목록에서 하나 또는 여러 개의 파일/폴더를 마우스로 클릭하여 드래그할 수 있다. 드래그가 시작되면 선택된 항목들은 반투명한 형태로 마우스 커서를 따라다닌다. 사용자가 드래그 중인 항목을 다른 폴더 위로 가져가면, 해당 폴더의 배경색이 하이라이트되어 드롭 가능한 영역임을 시각적으로 알려준다.</li><li><b>이동 로직:</b> 사용자가 드롭을 완료하면, 서버에 선택된 항목들의 부모 폴더 정보를 변경하는 API를 호출한다. 이동하려는 위치에 동일한 이름의 파일/폴더가 존재할 경우, "이미 같은 이름의 파일이 있습니다. 덮어쓰시겠습니까? 또는 이름 변경 후 이동하시겠습니까?" 옵션을 제공하는 다이얼로그를 표시한다. 대용량 파일(1GB 이상) 이동 시에는 실제 데이터 복사가 아닌 메타데이터만 변경하여 1초 이내에 작업이 완료되도록 한다.</li><li><b>컨텍스트 메뉴 이동:</b> 드래그 앤 드롭 외에도, 파일/폴더의 컨텍스트 메뉴에 '이동' 옵션을 제공한다. 이를 클릭하면 현재 폴더 구조를 보여주는 트리(Tree) 형태의 모달이 나타나며, 사용자는 대상 폴더를 선택하여 파일을 이동시킬 수 있다.</li></ul>
