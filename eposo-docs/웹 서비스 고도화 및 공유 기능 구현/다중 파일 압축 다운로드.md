# 다중 파일 압축 다운로드

<h3>Objective</h3><p>사용자가 여러 개의 파일이나 폴더를 한 번에 선택하여 하나의 압축 파일(ZIP)로 편리하게 다운로드할 수 있도록 한다.</p><h3>Key Results</h3><ul><li>총 1GB, 100개 파일 압축 및 다운로드 시작까지의 서버 처리 시간 15초 이내</li><li>압축 파일 생성 성공률 99.8%</li><li>생성된 ZIP 파일의 무결성(손상 없음) 100% 보장</li></ul><h3>기능 명세</h3><ul><li><b>사용자 인터페이스:</b> 파일 목록에서 사용자는 체크박스를 이용해 여러 파일 및 폴더를 선택할 수 있다. 2개 이상의 항목이 선택되면, 상단 액션 바에 '다운로드' 버튼이 활성화된다.</li><li><b>압축 프로세스:</b> 사용자가 '다운로드' 버튼을 클릭하면, 서버에 선택된 항목들의 목록을 전달하여 압축을 요청한다. 서버는 실시간으로 스트리밍 압축을 수행하여, 전체 압축이 완료될 때까지 기다리지 않고 생성되는 즉시 클라이언트로 데이터를 전송한다. 이를 통해 사용자의 대기 시간을 최소화한다. 압축 파일의 기본 이름은 "다운로드_YYYY-MM-DD.zip" 형식으로 생성된다. 선택된 항목들의 총 용량이 5GB를 초과할 경우, "대용량 파일은 압축 없이 개별 다운로드만 가능합니다."라는 안내를 표시한다.</li><li><b>진행 상태 알림:</b> 압축 및 다운로드 준비 과정 동안, 화면에 "파일을 압축하는 중입니다... (x/n)"와 같이 진행 상태를 나타내는 토스트 메시지를 표시하여 사용자가 현재 상황을 인지할 수 있도록 한다.</li></ul>
