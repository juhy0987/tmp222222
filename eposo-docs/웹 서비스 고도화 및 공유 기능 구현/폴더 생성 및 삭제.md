# 폴더 생성 및 삭제

<h3>Objective</h3><p>사용자가 파일을 체계적으로 정리할 수 있도록 직관적인 폴더 관리 기능을 제공한다.</p><h3>Key Results</h3><ul><li>'새 폴더' 생성 모달 팝업 시간 200ms 미만</li><li>폴더 생성/삭제 후 UI 반영 시간 500ms 미만</li><li>폴더 관련 기능 사용 중 에러 발생률 0.1% 미만</li></ul><h3>기능 명세</h3><ul><li><b>폴더 생성:</b> 파일 목록 영역 상단에 '새 폴더' 아이콘 버튼을 배치한다. 클릭 시, 현재 위치에 새로운 폴더를 생성하기 위한 입력 필드가 나타난다. 기본 폴더명은 "새 폴더"로 자동 지정되며, 동일한 이름의 폴더가 이미 존재할 경우 "새 폴더 (2)"와 같이 숫자 접미사를 붙여 제안한다. 사용자는 최대 50자까지 폴더명을 입력할 수 있으며, 파일 시스템에서 허용되지 않는 특수문자(`/  : * ? " < > |`)는 입력이 제한된다. Enter 키를 누르거나 포커스를 잃으면 폴더 생성이 확정되고, 0.5초 내에 UI에 반영된다.</li><li><b>폴더 삭제:</b> 사용자는 폴더의 컨텍스트 메뉴(우클릭) 또는 '더보기(...)' 메뉴를 통해 '삭제' 옵션을 선택할 수 있다. 삭제 실행 시, "폴더 내 모든 파일과 하위 폴더가 영구적으로 삭제됩니다. 정말 삭제하시겠습니까?"라는 내용의 확인 다이얼로그를 표시하여 사용자의 실수를 방지한다. '확인'을 누르면 삭제 API가 호출되며, 폴더 내 파일이 100개 이상일 경우 백그라운드에서 비동기적으로 처리하고 UI에서는 즉시 사라지도록 하여 사용자 경험을 저해하지 않는다.</li></ul>
