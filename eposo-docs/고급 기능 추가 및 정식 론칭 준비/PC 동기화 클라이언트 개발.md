# PC 동기화 클라이언트 개발

<h3>Objective</h3><p>사용자가 PC의 특정 폴더를 클라우드와 실시간으로 동기화하여, 웹 접속 없이도 로컬 환경에서 파일을 관리하고 자동으로 백업할 수 있도록 한다.</p><h3>Key Results</h3><ul><li>파일 변경(생성, 수정, 삭제) 후 클라우드 동기화 시작까지의 지연 시간 평균 5초 미만</li><li>클라이언트 프로그램의 CPU 점유율 평시 1% 미만, 동기화 시 15% 미만 유지</li><li>동기화 오류 발생률 0.01% 미만</li></ul><h3>기능 명세</h3><ul><li><b>설치 및 설정:</b> Windows 및 macOS를 지원하는 네이티브 설치 프로그램을 제공한다. 최초 실행 시, 사용자는 로그인 후 로컬 PC에서 동기화할 폴더를 지정한다. 지정된 폴더 내의 모든 파일과 하위 폴더 구조는 클라우드로 초기 업로드된다.</li><li><b>실시간 동기화:</b> 클라이언트는 지정된 폴더의 파일 시스템 이벤트를 지속적으로 감시(watching)한다. 파일 생성, 수정, 이름 변경, 삭제 등의 이벤트가 감지되면, 변경된 파일만 클라우드 서버와 동기화한다. 100MB 파일 기준, 수정된 부분만 비교하여 전송하는 델타 동기화(Delta Sync) 기술을 적용하여 네트워크 트래픽을 최소화한다.</li><li><b>UI/UX:</b> PC의 시스템 트레이(System Tray) 또는 메뉴 바(Menu Bar)에 아이콘 형태로 상주한다. 아이콘을 통해 동기화 상태(동기화 중, 완료, 오류)를 시각적으로 표시한다. 아이콘 클릭 시 최근 동기화된 파일 목록, 현재 동기화 진행률, 설정 메뉴 등을 포함하는 팝업 창을 제공한다. 파일 탐색기(Finder/Explorer)에서는 각 파일/폴더 아이콘 옆에 동기화 상태를 나타내는 오버레이 아이콘(체크, 동기화 중, 오류)을 표시한다.</li></ul>
